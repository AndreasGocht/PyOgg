Base Classes API
================

The class ``AudioFile`` acts as a base class for ``OpusFile``,
``VorbisFile`` and ``FlacFile``.  It provides common methods and
attributes.


AudioFile
---------

This class is Python-only and does not have any requirements for
specific shared libraries.
